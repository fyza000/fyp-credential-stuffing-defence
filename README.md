# fyp-credential-stuffing-defence
Final Year Project - BSc (Hons) Computer Security and Forensics
Student: Faiza Noor Noushad

##Project Overview
This project implements a lightweight, rule-based credential stuffing prevention system using behavioural risk scoring and adaptive authentication. The system analyses non-sensitive login signals to distinguish legitimate users from automated attacks and supplies graduated mitigation stratergies.

##Technology Stack
-FastAPI (Python)
-Redis
-SQLite / PostgreSQL
-Playwright / Selenium
-CAPTCHA and MFA (adaptive)
